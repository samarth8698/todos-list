# 📝 Todos List

A simple, responsive, and user-friendly Todo List application built using **React.js**. It allows users to manage their daily tasks efficiently by adding, viewing, completing, and deleting todos.

## 🚀 Features

- ➕ Add new todo
- 📋 View all todos
- ✅ Mark task as completed
- 🗑️ Delete todo
- ⚡ Fast and responsive UI
- 📱 Mobile-friendly interface
- ♻️ Component-based architecture

## 🛠️ Tech Stack

- React.js
- JavaScript (ES6)
- HTML5
- CSS3
- Create React App

## ⚙️ Installation

### Clone Repository

bash
git clone https://github.com/samarth8698/todos-list.git

### Navigate to Project

bash
cd todos-list

### Install Dependencies

bash
npm install

### Start Development Server

bash
npm start

Application runs on:

http://localhost:3000
